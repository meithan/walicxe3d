26 !> @brief Imposes a supernova remnant

29 !! NOTE: do not modify the code in this module! Import it in your user

32 !! How to use this module. To detonate a SNR:
33 !! 1) Import the 'snr' module in your user module
34 !! 2) Create an object of type snr_params_type to hold the parameters of
35 !! the SNR you want to detonate. The descriptions of the object's fields

46  ! This custom type contains the data required by the detonateSNR to
47  ! set off a supernova remnant.
48  ! The variables have the following meanings:
49  ! xc, yc, zc: physical coordinates of the SNR center
50  ! radius: initial radius of the SNR
51  ! mass: mass of SN ejecta
52  ! energy: total energy (thermal+kinetic) of explosion
53  ! chi: ratio of kinetic to total energy
54  ! -- The following fields are optional --
55  ! bx, by, bz: magnetic field strength in ejecta (may be ignored)
56  ! time: time at which the SNR will be imposed
57  ! metal: metallicity of the ejecta (may be ignored)
58  ! armed: a logical flag used to disarm the SNR once it is imposed
59  ! All physical values must be given in *CGS*.
60  type snr_params_type
61  real :: xc
62  real :: yc
63  real :: zc
64  real :: radius
65  real :: mass
66  real :: energy
67  real :: chi
68  real :: bx = 0.0
69  real :: by = 0.0
70  real :: bz = 0.0
71  real :: time = 0.0
72  real :: metal = 1.0
73  logical :: armed = .true.
74  end type snr_params_type

83  !! flow properties for the ejecta in a region of radius RSN centered at

215  !> @brief Simulates the remnant of a supernova explosion (SN Ia ejecta model)
216  !> @details Simulates the detonation of a supernova by imposing flow
217  !! properties following the SN Ia ejecta model of Jun & Norman (1996).
218  !! The method is similar to the 'simple' version, except for the following:
219  !! Density is taken as:
220  !! rho_c if r < r_c
221  !! rho_c*(r/r_c)^(-n) if r > r_c
222  !! where rho_c and r_c are the ensity and radius of the core,
223  !! and n (=7) is the core density power law index.
224  !! The core radius is given by:
225  !! r_c = R*[1-x*(3-n)*M/(4*pi*rho_0*R^3)]^[1/(3-n)]
226  !! while the core density by:
227  !! rho_c = 3*(1-x)*M/(4*pi*r_c^3)
228  !! where x (=3/7) is the mass fraction of the core, rho_0 is the density
229  !! at the edge of the core, and M and R are the total mass and radius of
230  !! the SN ejecta.
231  !! The velocity is taken as linearly proportional to radius:
232  !! v(r) = v0*r/R
233  !! where v0 is the velocity at the outer boundary and is given by:
234  !! v0 = Ek^(1/2)*{2*pi*rho_c*r_c^5/(5*R^2) +
235  !! + 2*pi*rho_0*R^3*[1-(R/r_c)^(n-5)]/(5-n)}
236  !! Finally, pressure can be obtained from the thermal energy and volume:
237  !! P = (1/CV)*Eth/(4/3*PI*R^3)
238  !! where Eth = E-Ek is the thermal energy (this is different from
239  !! the method of Jun & Norman).
240  !> @params snr_params A snrparams_type object containing the SNR parameters.
241  !! See the snr module main documentation for further details.
242  !> @param uvars Full flow variables array to be modified (U or UP)
